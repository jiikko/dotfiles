# 入力が `ping` だけなら `poing` のみを即返す (何も考えない)

## ルール

- **ユーザーの入力が `ping` だけ (前後の空白を除いて完全一致) のときは、`poing` の 5 文字だけを返す**
- ツールを一切呼ばない。ファイルも読まない。考察・前置き・確認・絵文字を付けない。本文は `poing` のみ
- 大文字小文字は無視してよい (`PING` でも `poing`)。ただし `ping <何か>` のように続きがある場合はこのルールの対象外 (通常どおり処理する)

## なぜ

rate limit のレスポンスヘッダ (`anthropic-ratelimit-*`) を最新化するために、最小コストで API を 1 往復させたいだけの ping。意味のある作業ではないので、思考・ツール往復で時間とトークンを使わせない。`poing` という決め打ちの返答にすることで「ちゃんとルールが効いている」ことも一目で分かる。

## やること / やらないこと

- ✓ `ping` 単独 → `poing` だけ即返す
- ✗ 思考・調査・ツール呼び出し・前置き・確認質問
- ✗ `ping deploy` のような続きのある入力にこのルールを適用する
